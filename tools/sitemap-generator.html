Create Your XML Sitemap

An XML sitemap lists a website's important pages, making sure search engines like Google can find and crawl them all. It can also help search engines understand your website structure. This tool helps you create a basic XML sitemap by manually providing a list of your URLs.

Note: This is a manual sitemap generator. It does not crawl your website to discover URLs. You must provide the list of URLs yourself.

How to Use This Manual Sitemap Generator

  1. Paste Your URLs: Enter the full URLs of your website's pages into the input area, with each URL on a new line. Ensure they are complete URLs (e.g., `https://www.example.com/your-page.html`).
  2. Set Optional Parameters (Optional):
    • Last Modified Date: You can specify a common last modification date for all URLs. If left blank, the `` tag will be omitted or set to the current date by some systems.
    • Change Frequency: Indicate how frequently the content of the pages is likely to change (e.g., daily, weekly, monthly). This is a hint for search engines.
    • Priority: Set a priority for your URLs, from 0.0 to 1.0. Higher values indicate more important pages. This is also a hint and may not be strictly followed by search engines.
    These optional parameters will be applied to all URLs in the list.
  3. Generate & Download: Click the "Generate Sitemap" button. The XML sitemap content will appear. You can then copy it or download it as a `sitemap.xml` file.

What is an XML Sitemap and Why Do I Need One?

An XML sitemap is a file that lists all the important URLs on your website. It helps search engine crawlers (like Googlebot) to more intelligently crawl your site, especially if:

  • Your site is very large.
  • Your site has a lot of archived content that may not be well-linked.
  • Your site is new and has few external links to it.
  • Your site uses rich media content (like video or images) or is shown in Google News.

While not a guarantee of better rankings, a sitemap helps ensure search engines are aware of all your content.

No, this is a manual sitemap generator. It does not crawl your website to automatically discover URLs. You need to provide the list of URLs that you want to include in your sitemap. For automatic crawling, you would typically use server-side tools or dedicated sitemap generation software.

These are optional tags in an XML sitemap:
  • <lastmod>: The date of last modification of the page. Search engines may use this information. Format: YYYY-MM-DD.
  • <changefreq>: How frequently the page is likely to change. Valid values: always, hourly, daily, weekly, monthly, yearly, never. This is a hint to crawlers.
  • <priority>: The priority of this URL relative to other URLs on your site. Valid values range from 0.0 to 1.0. Higher values suggest more important pages. This is also a hint.
This tool allows you to set these values uniformly for all listed URLs.

Once you have generated your `sitemap.xml` file:
  1. Upload it to the root directory of your website (e.g., `https://www.example.com/sitemap.xml`).
  2. Submit the sitemap URL to search engines like Google (via Google Search Console) and Bing (via Bing Webmaster Tools).
  3. You can also reference your sitemap location in your `robots.txt` file by adding a line like: `Sitemap: https://www.example.com/sitemap.xml`.
